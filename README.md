# 🌦️ Weather App

A simple and responsive Weather Application built using HTML, CSS and JavaScript. It fetches live weather data using the OpenWeather API.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Live temperature
- ☁️ Weather description
- 💧 Humidity
- 🌬️ Wind speed
- 🌤️ Weather icon
- ❌ Invalid city error handling
- 📱 Responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## ▶️ How to Run

1. Download or clone the project.

```
git clone https://github.com/your-username/Weather-App.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

4. Add your own OpenWeather API key in **script.js**.

```javascript
const apiKey = "YOUR_API_KEY";
```
